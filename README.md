zmailer
=======

a ZeroMQ actor to send emails !

Status 
------

Overview 
--------

 * Pure C
 * Developed following  [the CLASS language] (http://rfc.zeromq.org/spec:21)
 * Using zproject, zproto and Curl 

Content 
-------

Dependencies   
------------

 * libzmq, version > 4.2.0
 * czmq, version > 3.0
 * curl 

Building 
--------

    ./autogen.sh
    ./configure
    make
